#### TCN_classification

是根据An Empirical Evaluation of Generic Convolutional and Recurrent Networks for Sequence Modeling编写的

#### 亮点

+ 使用tf.data和tf.estimator编写
+ 支持tensorflow-serving部署
+ TCN是时间卷积序列，可以并行，速度快，可以与RNN、LSTM进行比较
